# harness-sdk

프로젝트 레포에 Karpathy LLM Wiki + GraphRAG + Harness SDKG 사상을 따르는 `.harness/`, `.claude/`, `CLAUDE.md` 구조를 자동 생성합니다.

harness-sdk/
├── reference/    명세 + default seed (수정 X)
├── src/          Python CLI 구현 (Stage 1~8)
